class dacapo.experiments.arraytypes.Mask

The type of data provided by an array. The ArrayType class helps to keep track of the semantic meaning of an Array. Additionally the ArrayType keeps track of metadata that is specific to this datatype such as num_classes for an annotated volume or channel names for intensity arrays.

class dacapo.experiments.arraytypes.ProbabilityArray

An array containing probabilities for each voxel. I.e. each voxel has a vector of length c where c is the number of classes. The l1 norm of this vector should always be 1. The class of this voxel can be determined by simply taking the argmax.
