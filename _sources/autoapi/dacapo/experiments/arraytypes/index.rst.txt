.. py:class:: DistanceArray




   An array containing signed distances to the nearest boundary voxel for a particular label class.
   Distances should be positive outside an object and negative inside an object.

   .. py:property:: interpolatable
      :type: bool


   .. py:attribute:: classes
      :type: Dict[int, str]

      


.. py:class:: Mask




   The type of data provided by an array. The ArrayType class helps to keep
   track of the semantic meaning of an Array. Additionally the ArrayType
   keeps track of metadata that is specific to this datatype such as
   num_classes for an annotated volume or channel names for intensity
   arrays.

   .. py:property:: interpolatable
      :type: bool



.. py:class:: EmbeddingArray




   A generic output of a model that could represent almost anything. Assumed to be
   float, interpolatable, and have sum number of channels.

   .. py:property:: interpolatable
      :type: bool


   .. py:attribute:: embedding_dims
      :type: int

      


.. py:class:: ProbabilityArray




   An array containing probabilities for each voxel. I.e. each voxel has a vector
   of length `c` where `c` is the number of classes. The l1 norm of this vector should
   always be 1. The class of this voxel can be determined by simply taking the
   argmax.

   .. py:property:: interpolatable
      :type: bool


   .. py:attribute:: classes
      :type: List[str]
